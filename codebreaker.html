A secret 4-digit code is generated each round. All 3 players guess against the same code at the same time — unlimited guesses, no turns. Each guess shows green if a digit is right digit & spot, yellow if it's in the code but the wrong spot, red if it's not in the code. First to crack it wins the round. First to win 2 rounds takes the match.
